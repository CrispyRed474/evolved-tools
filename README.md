# Evolved Floors Tools Hub

Static tools and applications for Evolved Floors customers and staff.

## Contents

- **Quote Tool** (`index.html`) — Interactive flooring quote calculator

## Deployment

This repository is deployed to Cloudflare Pages at **evolvedluxuryfloors.com.au**

### Local Testing

```bash
# Open index.html in a browser, or
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Build & Deployment

- **Build command:** None (static HTML)
- **Output directory:** `/` (root)
- **Deployment:** Cloudflare Pages (automatic on push to main)
